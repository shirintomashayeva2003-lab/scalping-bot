# 🤖 Scalping Signal Bot — Инструкция по запуску

## Что делает бот
- Сканирует 20 топ-ликвидных криптопар (BTC, ETH, SOL и др.)
- Анализирует RSI + MACD + EMA + Bollinger Bands + Объём
- Выдаёт сигнал: LONG 🟢 или SHORT 🔴
- Указывает таймфрейм (5м или 15м), вход, стоп-лосс, тейк-профит
- Держать сделку: 5-10 минут
- Автосканирование каждые 15 минут (по подписке)

---

## ШАГ 1 — Создать Telegram бота

1. Открой Telegram → найди **@BotFather**
2. Отправь `/newbot`
3. Придумай имя и username (например `ScalpingSignalBot`)
4. Скопируй токен — выглядит так: `1234567890:ABCdefGHIjklMNOpqrSTUVwxyz`

---

## ШАГ 2 — Запустить локально (на компьютере)

### Установка Python
Нужен Python 3.10+. Скачай с python.org

### Установка зависимостей
```bash
pip install -r requirements.txt
```

### Запуск
```bash
BOT_TOKEN="твой_токен_сюда" python bot.py
```

На Windows:
```
set BOT_TOKEN=твой_токен_сюда
python bot.py
```

---

## ШАГ 3 — Деплой на сервер (Railway — БЕСПЛАТНО)

### 3.1 Создай аккаунт
Зайди на **railway.app** → Sign Up (через GitHub)

### 3.2 Загрузи код
1. Создай репозиторий на GitHub, залей туда 3 файла:
   - `bot.py`
   - `analyzer.py`
   - `requirements.txt`

2. В Railway нажми **New Project** → **Deploy from GitHub repo**
3. Выбери свой репозиторий

### 3.3 Добавь переменную окружения
В Railway → Variables → добавь:
```
BOT_TOKEN = твой_токен_от_BotFather
```

### 3.4 Добавь файл запуска
Создай файл `Procfile` (без расширения):
```
worker: python bot.py
```

Готово! Бот будет работать 24/7 бесплатно.

|---------|----------|

## ⚠️ Важно

- Бот работает с данными **Binance** (через публичный API, бесплатно)
- В **Tiger Broker** вручную открываешь сделку по сигналу
- Сигнал — это рекомендация, не гарантия прибыли
- Всегда используй стоп-лосс
- Не рискуй более 2-3% от депозита на одну сделку
